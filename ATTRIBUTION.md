# Attribution — Formula 67

This file lists external references, generated assets, and third-party
libraries used in the project. It is intended to make the asset sources clear
for project review.

## Car sprites (player + AI)

Files in `imgs/`: **red-car.png**, **green-car.png**, **purple-car.png**, **white-car.png**, **grey-car.png**.

The car sprites were created with an AI-assisted image tool from original
prompts, then exported as PNG files and adjusted for use in the game. They were
not copied from a downloadable car sprite pack.

## Other bitmaps in `imgs/`

The following bitmap assets were based on online references. Some files were
cropped, resized, or recolored to match the game style.

| File | Use in project | Reference |
|------|----------------|-----------|
| `grass.png` | Background texture | <https://www.shutterstock.com/th/search/grass-game-texture?dd_referrer=https%3A%2F%2Fwww.google.com%2F> |
| `track.png` | Road surface | <https://www.pngwing.com/en/search?q=race+track> |
| `track-border.png` | Border and collision mask source | <https://www.pngwing.com/en/search?q=race+track> |
| `finish.png` | Checkered finish strip | <https://www.freepik.com/premium-vector/background-black-white-squares-chess-board-vector_26468292.htm> |

Any later edits, recoloring, or AI-assisted adjustments were made only to adapt
the assets to the game presentation. The original reference links remain listed
above.

## Generated by Project Code

- Files under `reports/` are generated from `visualize.py`, matplotlib, and
  `stats/*.csv`.
- Menu, HUD, results panels, podium graphics, and in-game interface elements
  are drawn with pygame code.

## Libraries

See [requirements.txt](requirements.txt). The main libraries are pygame,
matplotlib, numpy, and pandas.
